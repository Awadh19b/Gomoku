# Go-Moku Game

A simple implementation of the Go-Moku (Gomoku) game in Java, featuring a command-line interface with AI opponent.

## Game Rules
- Play against an AI opponent on an 8x8 board
- First to get 5 in a row (horizontally, vertically, or diagonally) wins
- You play as 'X' and the AI plays as 'O'

## Prerequisites
- Java 11 or higher
- Maven 3.6.0 or higher
